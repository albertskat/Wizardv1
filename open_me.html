Step 1: Getting Started

* Find a favorite text editor. I recommend Atom or Brackets. (It's possible to work directly within Northpass but there is more functionality within an editor.)
* Open your text editor and create a blank file. Name it with a .html file extension. Ex: CTE-100 Tunaverse POA.html

Step 2: Choose How Many Cards You Want

* Copy the code you want and paste it into your new file.
* Change the title in TWO places on each card.
* Add a description for the front of each card, or delete Card # Description Here to have no description.
* Replace each fa-file with the font awesome icon you want for the front of each card. Link opens in new window.

NOTE: If you have a Card Title longer than 2-3 words and it is pushing the bottom of the card downward, change the 2nd title from h1 to h2. This will shrink the size of the font for the title.

Step 5: Extra Formatting

* Making font bold: <strong>Text here</strong>

* Making font SPS Blue: <span style="color: rgb(0, 170, 229);">Text here</span>
NOTE: See the SPS Pattern Library for more SPS branded colors.

* Changing color of icons: Add sps- plus the color you want before the word icon in the span classes.
Example: <span class="sps-orange icon fa fa-thumbs-up fa-3x fa-fw"></span>

* Changing font size: Coming soon.

* Adding a line break: <br>

* Underlining: <u>Text here</u>

* Exclamation (!!!) icon: <span class="icon fa fa-exclamation-circle fa-1x fa-fw"></span>

* Lightbulb icon: <span class="icon fa fa-lightbulb fa-1x fa-fw"></span>

Step 6: Finishing Up

* When finished, make sure your .html file is saved in the Shared Drive.
* Copy and paste the ENTIRE code from your new file into the text editor in Northpass. (Make sure you're on plain text editing mode (HTML) and not rich text editing mode)
